Mandelbrot and friends

Some scrappy Mandelbrot visualisations for London Computation Club:

  • the Mandelbrot set itself (click to zoom);
  • the Julia sets for each point of the Mandelbrot set (mouseover to explore different points);
  • escape time visualisation for the Mandelbrot set (click to add particles, press C to clear, press space to add random particles).
